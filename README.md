# 人民城轨2.0 - 运营管理后台

🚇 **专业的城轨运营管理系统前端Demo** 

一个功能完整、设计精美的运营管理后台系统，专为城市轨道交通运营管理打造。

## ✨ 系统特色

- 🎯 **二级导航架构** - 清晰的功能模块分类组织
- 🔐 **用户认证系统** - 完整的登录认证流程
- 📱 **响应式设计** - 支持桌面端和移动端
- ⚡ **模块化架构** - 组件化设计，易于维护扩展
- 🛡️ **等保三级合规** - 符合等保三级安全要求
- 🔧 **完善错误处理** - 强大的错误边界和异常恢复

## 🏗️ 系统架构

```

│   ├── 展位管理          # 展会展位信息管理
│   └── 直播管理          # 直播内容管理

│   ├── 日活、月活等统计  # 核心运营指标统计
│   ├── 用户行为统计      # 用户行为数据分析
│   ├── 作品数据统计      # 内容浏览、点赞、评论等统计
│   ├── 用户画像分析      # 用户标签分群与画像
│   └── 流量分配配置      # 推荐算法与流量分配
└── 系统管理             
    ├── 用户管理          # 用户信息查询与管理
    ├── 权限管理          # 管理员账号和权限管理
    ├── 消息管理          # APP系统消息推送管理

    ├── 日志管理          # 等保三级合规日志管理
```
