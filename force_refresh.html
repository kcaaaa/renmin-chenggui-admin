📋 手动刷新指南

  1. Windows: 按 Ctrl + F5 或 Ctrl + Shift + R
  2. Mac: 按 Cmd + Shift + R
  3. 手机: 长按刷新按钮，选择"硬重新加载"
  4. 彻底清除: 打开开发者工具 → Network → 勾选"Disable cache" → 刷新

🛠️ 浏览器缓存清除方法

  1. 按 Ctrl + Shift + Delete (Windows) 或 Cmd + Shift + Delete (Mac)
  2. 选择清除"缓存的图片和文件"
  3. 选择时间范围为"最近1小时"或"全部时间"
  4. 点击"清除数据"按钮
  5. 重新访问页面

如果仍然遇到问题，请尝试使用浏览器的隐私/无痕模式访问，或联系技术支持。
